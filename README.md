# C From Zero

> Documenting my journey of learning C programming from scratch.

This repository is a public record of my journey in learning the C programming language as a **BS Applied AI & Data Science student at the Indian Institute of Technology Jodhpur (IIT Jodhpur)**.

Rather than only collecting solutions, this repository is designed to document the learning process—from writing my first `Hello, World!` program to understanding memory management, pointers, data structures, and building complete applications.

My goal is to write clean, readable, and well-documented code while building a strong foundation in systems programming and computer science.

---

## Why this repository?

I believe learning is more valuable when it is documented.

This repository serves as:

* My personal learning journal
* A collection of C programming examples
* Notes for revision
* Practice exercises
* Mini projects
* A resource for anyone beginning their C programming journey
